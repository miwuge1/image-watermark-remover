# Image Watermark Remover - 需求文档

## 项目概述

**项目名称**：Image Watermark Remover  
**项目类型**：Python 图像处理工具  
**核心功能**：去除图片中的水印，使用 AI 修复技术填充被水印覆盖的区域

---

## 功能需求

### 1. 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 单图去水印 | 输入带水印图片，输出无水印图片 | P0 |
| 批量处理 | 批量处理文件夹内所有图片 | P1 |
| 自定义蒙版 | 支持用户指定水印区域蒙版 | P1 |
| CLI 命令行 | 支持命令行参数操作 | P0 |

### 2. 技术实现

- **算法**：OpenCV 图像修复（Inpainting）
  - TELEPHOTO 算法：适合高质量修复
  - NS 算法：适合快速处理
- **输入格式**：JPG, PNG, BMP, WEBP
- **输出格式**：与输入格式一致

### 3. 用户交互

#### CLI 模式

```bash
# 基本用法
python main.py input.jpg output.jpg

# 使用自定义蒙版
python main.py input.jpg output.jpg --mask watermark_mask.png

# 批量处理
python main.py input_dir output_dir --batch
```

#### Python API 模式

```python
from watermark_remover import remove_watermark

remove_watermark("input.jpg", "output.jpg")
remove_watermark("input.jpg", "output.jpg", mask_path="mask.png")
```

---

## 性能要求

- 单图处理（1920x1080）：< 3 秒
- 批量处理：支持文件夹并发处理
- 内存占用：< 500MB（单图）
